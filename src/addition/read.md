# Суммирование массива с использованием многопоточности

## Цель проекта

Цель этого проекта — демонстрация базовых принципов многопоточного программирования на Java через задачу суммирования массива. Мы создадим массив большого размера, заполненный случайными числами, и два потока, которые будут суммировать каждую половину массива. После завершения потоков, итоговая сумма будет выведена в консоль.

## Технические требования

- Язык программирования: Java
- Версия JDK: 8 или выше

## Функциональные требования

1. **Инициализация массива**: Создать массив целых чисел размером, например, 1 000 000 элементов. Элементы массива должны быть заполнены случайными числами.

2. **Создание потоков**: Создать два потока (`Thread`), каждый из которых будет суммировать свою половину массива.

3. **Суммирование**: Первый поток суммирует первую половину массива, второй поток — вторую половину.

4. **Синхронизация потоков**: Использовать метод `join()` для того, чтобы основной поток ожидал завершения вычислительных потоков.

5. **Вывод результата**: После завершения всех потоков, сложить результаты каждого потока и вывести итоговую сумму в консоль.

### Краткое описание классов

- `Main`: Основной класс, который содержит метод `main`. В этом классе будет инициализирован массив и созданы потоки.

- `SumThread`: Класс, который расширяет `Thread` или реализует `Runnable`. Содержит логику суммирования элементов массива.

## Примерный план выполнения

1. Создать и заполнить массив случайными числами.
2. Написать класс `SumThread`, который суммирует часть массива.
3. В классе `Main` создать и запустить два потока, передав им соответствующие части массива для суммирования.
4. Использовать `join()` для ожидания завершения потоков.
5. Сложить результаты и вывести итоговую сумму.

## Критерии успешности

- Код компилируется и успешно выполняет задачу суммирования.
- Оба потока корректно суммируют свои части массива.
- Итоговая сумма корректно выводится в консоль после завершения всех потоков.
